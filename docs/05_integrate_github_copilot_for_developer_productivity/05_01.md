# Task 01 - Use GitHub Copilot to create an Issue

## Introduction

Zava's Storefront has been upgraded from .NET 6 to .NET 8 (LTS). Since Zava as an organization is committed to improving developer efficiency with AI, this upgrade serves as a reference implementation demonstrating how Copilot can create issues and automate version updates.

This lab will show you an example of how to create issues for upgrades and assign them to Copilot for implementation. 

## Description

In this task you will review the .NET 8 upgrade that was completed, and learn how to create similar issues for future updates using GitHub Copilot.

## Success Criteria

- You have reviewed the .NET 8 upgrade implementation
- You understand how to create issues using GitHub Copilot in Visual Studio Code for similar upgrades

## Learning Resources

- [Using GitHub Copilot to create or update issues](https://docs.github.com/en/copilot/how-tos/use-copilot-for-common-tasks/use-copilot-to-create-or-update-issues){:target="_blank"}

## Key Tasks

### 01: Review the completed upgrade

Review the .NET 8 upgrade that has been completed in this repository.

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. Navigate to your forked repository and review the files that were updated for the .NET 8 upgrade:
   - `src/ZavaStorefront.csproj` - Updated TargetFramework to net8.0
   - `src/Dockerfile` - Updated base images to .NET 8 SDK and runtime
   - Documentation files - Updated references to .NET 8

2. Review the commit history to see how the upgrade was implemented.

</details>

### 02: Learn to create similar issues (Optional)

You can practice creating issues for future upgrades using GitHub Copilot Chat.

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. Navigate to your forked repository for this lab at GitHub and select Chat with Copilot from the top menu:

    ![Chat with Copilot Icon](../../media/0201_chatcopilot.png)

2. Confirm that you have the correct repository context. If you started from the repository from Chat, you should see it selected:

    ![Chat with Copilot dialog](../../media/0201_chatcopilot_2.png)

3. You can practice creating issues for other potential upgrades or features. For example:

    ```bash
    /create-issue Update NuGet packages to latest versions
    ```

</details>
