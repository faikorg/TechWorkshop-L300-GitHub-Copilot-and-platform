# Task 02 - Review the .NET 8 upgrade implementation

## Introduction

Zava's Storefront has been upgraded from .NET 6 to .NET 8 (LTS). Since Zava as an organization is committed to improving developer efficiency with AI, this serves as a reference implementation showing how Copilot can automate version updates.

In this task you will review the completed upgrade and understand how Copilot can be used to implement similar updates. 

## Description

In this task you will review the .NET 8 upgrade implementation and verify that the solution builds and runs correctly.

## Success Criteria

- You have reviewed the .NET 8 upgrade changes
- You have verified the solution builds and runs on .NET 8

## Key Tasks

### 01: Review the .NET 8 upgrade changes

Review the changes that were made to upgrade the solution to .NET 8 (LTS).

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. In your local repository, review the following files that were updated:
   
   - `src/ZavaStorefront.csproj` - Check the TargetFramework property
   - `src/Dockerfile` - Review the base image versions
   - `src/README.md` - See the updated technology stack documentation
   - `docs/01_development_environment_setup/01_03.md` - Installation instructions

2. Use git to view the changes:
   
   ```bash
   git log --oneline --grep="NET" -n 5
   ```

</details>

### 02: Build and test the upgraded solution

Verify that the solution builds and runs correctly with .NET 8.

<details markdown="block">
<summary><strong>Expand this section for detailed steps</strong></summary>

1. Navigate to the `src` directory:

   ```bash
   cd src
   ```

2. Restore dependencies:

   ```bash
   dotnet restore
   ```

3. Build the solution:

   ```bash
   dotnet build
   ```

4. Run the application:

   ```bash
   dotnet run --project ZavaStorefront.csproj
   ```

5. Open your browser and navigate to the URL shown in the terminal (e.g., `https://localhost:7060`) to verify the application runs correctly.

6. Stop the application with Ctrl+C.

</details>

## Summary

You've completed this task. You have reviewed the .NET 8 (LTS) upgrade implementation and verified that the solution builds and runs correctly.
